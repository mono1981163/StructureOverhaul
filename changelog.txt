VaultEagle Console
******************

Version 1.2
===========
- 2017 version
- It is no longer a console, but a exe that can be run with commands. (Can still be run from Command Line)
- A new command was added -configuration=filePath (file path can be a local path or vault path).
 - Allows to run the program with extra settings. See synchronizersettings.txt for more info.
